redis数据库安装

参考https://www.cnblogs.com/silent2012/p/3499654.html
按着这个博主的步骤安装redis数据库
里面有个需要注意的地方
1.现版本的redis.conf是在主目录下,不在src目录下.
2.默认端口6379
3.启动这样操作./redis-server ./redis.conf & (&是在后台运行，redis默认会在前台运行，需要注意)
失败了尝试给文件夹加上足够权限，或者可能是路径问题.

然后安scrapy到下班也没弄好,感觉我这个版本的centos有点问题，照着百度前20多个答案安了各种包 都解决不了. gcc编译仍然有问题..

发现了另外一篇黑科技代理池的文章..windows下就能用的,很方便了

https://github.com/qiyeboy/IPProxyPool
http://www.cnblogs.com/qiyeboy/p/5693128.html
照着操作就行了,就一点小问题,python3下的话可能会稍微有点问题
web.py必须去官网下载
最后解压后python setup.py install
然后安装到0.40就会报错，但其实是成功安装了的
好像这种姿势也可以 pip install web.py==0.40.dev0
然后文章里还少提了一个psutil库的安装,
pip install psutil即可.
这个代理池除了直接 python IPProxy.py运行外,还可以对我们的爬虫提供API,
不爬虫使用的话,我们可以直接在浏览器里面找
http://localhost:8000/
第一个参数是IP,第二个参数是端口号,第三个参数是分数,就是越稳定分数越高.
然后直接在IE浏览器里改代理就行了

今后再也不怕封IP了2333
